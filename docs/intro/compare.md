## 页面路由导航

路由导航，统一使用如下 API：
* `$api.navigateBack`
* `$api.navigateTo`
* `$api.redirectTo`

为了保证各个平台导航一致性，建议全部用绝对路径，即导航 url 以 `/` 开头，一方面路径更清晰，其次可以更好兼容 `快应用`。
